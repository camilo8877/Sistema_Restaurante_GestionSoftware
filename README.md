# Sistema de Gestión de Restaurante

## Descripción

Aplicación web desarrollada con Flask para la gestión de restaurantes.

## Funcionalidades

- Autenticación de usuarios
- Gestión de menú digital
- Gestión de pedidos
- Reservas
- Reportes
- Facturación básica

## Tecnologías

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Instalación Local

### Clonar repositorio

git clone https://github.com/usuario/restaurant_system.git

### Entrar al proyecto

cd restaurant_system

### Crear entorno virtual

python -m venv venv

### Activar entorno virtual

Windows:
venv\\Scripts\\activate

### Instalar dependencias

pip install -r requirements.txt

### Ejecutar proyecto

flask run
